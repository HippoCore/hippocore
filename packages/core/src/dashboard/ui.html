Connect Hippo Core to your agent frameworks. Once connected, agents automatically get hippo_recall and hippo_remember tools.

Add to Any Agent (AGENTS.md / CLAUDE.md)

Add these instructions to your agent's instruction file:

## Memory (Hippo Core)
At the START of every task, call hippo_recall with the current task description.
At the END of every task, call hippo_remember to store what was learned.
Never ask the user to use these tools — call them automatically every time.
Manual MCP Config
{
  "mcpServers": {
    "hippo-core": {
      "command": "npx",
      "args": ["@hippo-core/core", "mcp"]
    }
  }
}

Windows: use "command": "cmd" with args ["/c", "npx", "@hippo-core/core", "mcp"]
