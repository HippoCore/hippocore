The problem

Every AI agent starts
from zero every time.

Without memory, agents are expensive, impersonal, and stateless. They can't learn from past interactions, build relationships, or improve over time. You're paying for the same context window over and over.

Without Hippo Core
// Every request starts from scratch
// Full conversation history sent every time
// 3,000+ tokens per request
// Agent has no idea who you are

const response = await agent.run(
  "What mortgage should I get?"
);

// "I'd be happy to help! Could you tell
// me your budget and preferences?"
// (for the 47th time)
With Hippo Core
// Memories retrieved, context injected
// ~400 tokens per request
// Agent knows exactly who you are

const memory = createMemory({ apiKey });
const { systemPrompt } = await memory.before(
  userId, userMessage
);

// "Based on your $500k budget and preference
// for fixed rates, I'd recommend..."

How it works

Four steps. Two lines of code.

Hippo Core sits between your user and your agent. It handles everything.

01
Store
Every interaction is embedded and stored in a local SQLite file. AI extracts facts, preferences, and intent automatically.
→
02
Retrieve
Semantic similarity search finds the most relevant memories for each new request. Ranked by relevance + importance score.
→
03
Inject
Top memories are injected into the system prompt within a configurable token budget. Never exceeds your limits.
→
04
Improve
Feedback signals boost or suppress memories. Important memories surface more. Noise is compressed away daily.

Features

Production-ready from day one.

🧠
Semantic Retrieval
Vector embeddings with cosine similarity. Memories ranked by relevance blended with importance score. Not just recency.
⚡
Token-Aware Injection
Hard token budget enforced on every request. Highest-ranked memories in first. Never blows up your context window.
📦
Zero Infrastructure
Single SQLite file. No Docker, no Postgres, no cloud account. Works on any machine with one npm install.
🔌
Framework Adapters
Native adapters for OpenClaw and Paperclip. Generic middleware for any other framework. Drop-in, not a rewrite.
🔒
Local First
All memory stays on your machine. Works fully offline with Ollama. No data sent anywhere except the LLM calls you configure.
📊
Developer Dashboard
Built-in monitoring at localhost:4444. Token savings, memory inspector, retrieval debug, prompt preview. npx @hippo-core/core dashboard
🔄
Session Buffer
Short-term session history kept separate from long-term memory. Agents handle "what did I just say?" correctly.
📉
Auto Compression
Daily job merges short-term memories into long-term profiles. Prunes noise. Decays stale importance scores automatically.
👍
Feedback Loop
Mark memories as helpful or harmful. Importance scores adjust over time. The more you use it, the smarter it gets.

Quick start

Up and running in 2 minutes.

// 1. Install // npm install @hippo-core/core import { createMemory } from '@hippo-core/core'; const memory = createMemory({ apiKey: process.env.OPENAI_API_KEY, // or point at Ollama, OpenRouter, Groq... }); async function runAgent(userId, userMessage) { // 2. Retrieve memories + build enriched prompt const { systemPrompt, tokenStats } = await memory.before( userId, userMessage, 'You are a helpful assistant.' ); // 3. Run your agent with memory context injected const response = await yourAgent(systemPrompt, userMessage); // 4. Store the interaction await memory.after(userId, userMessage, response); return response; }
import { Agent } from 'openclaw'; import { withMemory } from '@hippo-core/core/adapters/openclaw'; // Wrap your existing agent — one line const agent = withMemory(new Agent({ model: 'gpt-4o', systemPrompt: 'You are a mortgage advisor.', tools: [searchTool, calcTool], })); // Use exactly as before — memory is automatic const result = await agent.run('user_123', userMessage);
import { createAgent } from 'paperclip-ai'; import { memoryPlugin } from '@hippo-core/core/adapters/paperclip'; const agent = createAgent({ model: 'gpt-4o' }); // Register the plugin — that's it agent.use(memoryPlugin()); const result = await agent.run({ userId: 'user_123', message: 'What mortgage fits my budget?', }); // Agent now has full memory context
